Welcome to the Hang Project

A production-style real-time messaging system built with FastAPI, WebSockets, Redis Pub/Sub, PostgreSQL, and Dockerized multi-instance deployment behind Nginx load balancing. This project demonstrates a scalable distributed backend architecture for real-time communication systems, not just a basic chat application.

Technologies

HTML, CSS, Jinja, JavaScript, PostgreSQL, Python, RESTful, NGINX, Docker, Redis, FastAPI

Welcome to the Durden Project

Durden is a production-style object detection API built with FastAPI and YOLO, supporting multiple models, structured predictions, and asynchronous model loading. It's containerized and deployed on Google Cloud Run with automated GitHub Actions CI/CD for testing, Docker image publishing, and deployment.

Technologies

Python, FastAPI, YOLO, Docker, Google Cloud Run, GitHub Actions, CI/CD

Welcome to the Transcendence Project

Transcendence is a full-stack web development project focused on building a real-time multiplayer web application with a modern client-server architecture. The project revolves around recreating and extending the classic Pong game into an online platform featuring user authentication, matchmaking, live gameplay, and social interaction features.

Technologies

HTML, CSS, JavaScript, PostgreSQL, Django, RESTful, WebSocket, Daphne, Redis, NGINX, Docker, Make

Welcome to the Inception Project

Inception is a containerized web infrastructure project focused on designing and deploying a secure, production-like environment using Docker. The project consists of a multi-service architecture where NGINX serves as a TLS-enabled reverse proxy, WordPress provides the application layer, and MariaDB manages persistent data storage. Each service is built from custom Docker images and runs in an isolated container connected through a dedicated Docker network.

Technologies

Docker, NGINX, Wordpress, MariaDB, VMBox

Welcome to the IRC Project

IRC (Internet Relay Chat) is a versatile and powerful messaging project that facilitates real-time communication among users across the globe. Designed with a focus on community building and connectivity, IRC supports multiple channels, private messaging, and file sharing, making it an indispensable tool for online collaboration and social interaction. With its rich history and enduring popularity, IRC fosters a sense of belonging, allowing users to engage in meaningful conversations and share information seamlessly. This project showcases the potential of online communication and its role in bringing people together, regardless of physical distance.

Technologies

C++, WebSocket, STL, KVIrc, WeeChat, Make, CMake
